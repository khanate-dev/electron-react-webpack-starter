# CBL Garment Tracking

React Electron desktop application for CBL Garment Tracking

## Building Blocks

- `Electron` with `Electron Forge`
- `React`
- `Typescript`
- `React Router`
- `MUI 5`
- Formatting with `Prettier`
- Linting with `ESLint`
- `pnpm` for package management
